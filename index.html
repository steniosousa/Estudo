O que é JavaScript?

JavaScript é uma linguagem de script orientada a objetos, multiplataforma. É uma linguagem pequena e leve. Dentro de um ambiente de host (por exemplo, um navegador web) o JavaScript pode ser ligado aos objetos deste ambiente para prover um controle programático sobre eles. JavaScript tem uma biblioteca padrão de objetos, como: Array, Date, e Math, e um conjunto de elementos que formam o núcleo da linguagem, tais como: operadores, estruturas de controle e declarações. O que é HTML?

HTML abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web.

Ao acessar uma página web através de um navegador, ele é capaz de interpretar o código HTML e renderizá-lo de forma compreensível para o usuário final, exibindo textos, botões etc. com as configurações definidas por meio das diversas tags que essa linguagem dispõe. Atualmente a HTML encontra-se na versão 5 e é padronizada pelo W3C (World Wide Web Consortium), uma organização internacional responsável por estabelecer padrões para a internet, como a linguagem XML, CSS e o SOAP.

Estrutura básica

Todo documento HTML é delimitado pela tag < html >. Sendo assim, todo o conteúdo do seu documento deve estar entre < html > e < /html > .

Head

Além disso, o cabeçalho é delimitado pela tag < head >. No cabeçalho ficam as principais “configurações” do seu documento. Aqui ficam as chamadas para arquivos externos (tag < link >), título da sua página, configurações de tags para mecanismos de busca (meta tags), entre outras coisas. O conteúdo do cabeçalho deve estar entre < head > e < /head > .

Body

Por fim, o corpo da sua página, é delimitado pela tag < body > . No corpo do documento, ficará todo o conteúdo que de fato será “visível” para o usuário que acessar sua página. Aqui fica a sua logomarca, sua barra de navegação, texto do seu blog, etc. O conteúdo do corpo deve estar entre < body > e < / body > .
